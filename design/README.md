# Design source files

This folder holds original design scratch and reference art that is **not**
shipped or imported by the application. It exists to keep these source files
intentionally archived without cluttering the repository root.

These files are not referenced by `src/`, `server/`, or `public/`. If you need
an asset to be served by the app, add the production-ready (optimized) copy
under `public/assets/` instead — see the asset convention in the root
[`README.md`](../README.md).

## Contents

- `Joustur-board.png`, `Joustur WIP.png` — Joustur Skatur™ board reference art
- `Main Caracts with Photos.pdf` — character reference sheet
- `punch-skater.png`, `apprentice.png`, `carbon-street.png` — original
  (unoptimized) character/board art; optimized copies live under `public/`
- `Screenshot 2026-05-19 213048.png` — design reference screenshot
- `fin~2.mp4` — promotional/animation source clip
