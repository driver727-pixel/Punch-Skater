# Weapon Assets

Place transparent PNG weapon images here for use in the Card Forge weapon layer.

Requirements:
- Format: PNG with transparent background (no solid background)
- Recommended size: 400×400px or larger
- Naming convention: lowercase-hyphenated (e.g., `hockey-stick.png`, `road-sign.png`)

## Registered weapons (in `src/pages/cardForge/constants.ts`):
- `hockey-stick.png` — Hockey Stick
- `road-sign.png` — Street Sign
- `crutch-blue.png` — Crutch Lance
- `medieval-lance.png` — Medieval Spear

To add new weapons, place the image here and add an entry to the `WEAPON_ASSETS` array in constants.
